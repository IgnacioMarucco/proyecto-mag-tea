# ADR-0001: Java 21 como versión de la JVM

## Contexto

Spring Boot 4.x requiere Java 21 como mínimo.

## Decisión

Se usa **Java 21** (LTS, soporte hasta 2031).

## Alternativas consideradas

**Java 17** — descartado porque Spring Boot 4.x no es compatible.

## Consecuencias

**Positivas:**
- Único LTS compatible con Spring Boot 4.x
- Virtual Threads (Project Loom): miles de threads concurrentes con bajo overhead
- Records maduros: DTOs compactos e inmutables sin Lombok
- Pattern matching estable: código más expresivo en lógica de dominio

**Negativas / trade-offs:**
- Diferencia con lo visto en Java 17 — requiere aprendizaje de features nuevos
